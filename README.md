# Linkedin-Network-Behavior Analysis


**Objective**
To evaluate whether company size influences Linkdin connection strength.

**Data Processing**
1. Cleaned and converted numeric fields.
2. Removed missing values
3. Created company size categories
4. Segmented professionals into network tiers


**Analysis**
1. Correlation analysis (r=0.04)
2. Category-based comparison
3. Distribution analysis
4. Network tier segmentation.

**Key Insights**
Professional networking strength is behavior-driven rather than organization-driven.

**Tools used**
Pythin, Pandas, Matplotlib, Jupyter Notebook
